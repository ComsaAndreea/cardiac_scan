# Cardiac MRI Segmentation using Deep Learning

## Overview

This project implements a complete deep learning pipeline for automated segmentation and quantitative analysis of cardiac Magnetic Resonance Imaging (MRI) data.

The system focuses on **Short-Axis (SAX) Cardiac MRI images acquired during the End-Diastolic (ED) phase** and performs multi-class segmentation of:
- Left Ventricle (LV)
- Right Ventricle (RV)
- Myocardium (MYO)

using:
- U-Net
- Attention U-Net
- U-Net++

---

# Dataset Creation

A custom dataset named **CombinedSAX_ED** was generated by combining:
- ACDC
- M&Ms / M&Ms-2

Only SAX End-Diastolic images were retained.

Dataset generation included:
1. ED Frame Extraction
2. Label Standardization
3. View Filtering
4. Dataset Merging
5. Dataset Splitting
